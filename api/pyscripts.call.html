call¶

pyscripts.call(args, drop=None, call_name='func')[source]¶

Process the callable after dropping some values in the arguments. The callable is expected to take the remaining values.

Parameters:
  • args (argparse.Namespace) – arguments obtained from the parser.
  • drop (list(str) or None) – values to drop.
  • call_name (str) – name of the callable in the arguments.
Returns:

whatever is returned in the callable call.
